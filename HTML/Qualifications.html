Job Specifications
On this page, you will find all the information on job specifications as a web developer as well as how to kick-start your application if you want to apply to work for us. Here is a list of requirements and responsibilities as a web developer.
 
Web Developer Requirements
The list below conisists of requirements that a web developer must be able to fulfill to be successful:  
  - Bachelor's degree in Web Developement or any relted field or relevant experience in web development
  - Cab be a dedicated team player
  - The ability to thrive in a work environment that can be fast paced
  - The ability to demonstrate sufficient knowledge in programming language
  - Adequate knowledge and previous experience in programming
  - Be able to make independant choices
  - Be able to solve complex problems
  - Must be able to express and develop ideas clearly
  - Confident in using HTML, CSS and JavaScript
  - Requires good written and verbal communication skills 
 
 
Web Developer Responsibilities
As with any job, there are reponsibilities involved.Here is a list of some responsibilities as a web developer:
   -  Working with teams to resolve conflicts, develop content criteria, prioritize requiremnets and find and agree on a solution.
  - Identify errors through testing or customer feedback and rectify any errors
  - Writing, designing and editing website content and directing other team members that are producing content for a website
  - Designing, building, testing and maintaining a website or software application
  - Backing up files from websites to local directories so that they can be recovered
  - Performing and directing any website updates
  - Making sure that code meets industry standards and is comatiable with a range of web browsers, operating systems and devices
