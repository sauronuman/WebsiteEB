Web Courses
 
At Hot Beans we encourage you to always be learning. Here we will list web courses available for different skill types to test yourself and improve your knowledge. Being successful within these courses could lead to a job within our business or these courses can take you wherever you wish to go.
 
 
Beginner Courses
The courses listed below are courses for people new to web development wanting to understand the basics. Here are some sites we think you would love to learn from.
 
Codecademy offers a course Build a website with HTML for beginners. It takes 10 weeks to complete and there are no prior requirements for it. Once you have successfully completed the course you will receive a certificate of completion.
One of the best ways to start coding is by building websites. whether you want to tweak your business's site, hone your web developement skills, or learn to collaborate with developers, this skill path will help you get there. Learn important HTML fundamentals and practice your new skills with real-world projects.
Skills you will gain:
 - Read and write basic HTML
 - Implement basic web design principles
  - Build a static website
 
Intermediate Courses
These courses are for people who already understand basics of web development and have an aspiration to learn new techniques and develop their knowledge. it may quiz you too to test your knowledge! Listed below are some sites that could help you push your knowledge.
 
Another course Codecademy offers for a slightly higher skill set Learn Intermediate CSS. Learn how to implement elegant transitions, create new layouts and serve users with dynamic ends. It takes approximately only 15 hours to complete. It is advised before you do this course you complete Learn HTML and Learn CSS which are beginner courses on codecademy.
What you will gain:
 - Implement elegant transitions
 - Create new layouts
 - Give website flexibility to serve users with diffrent window dimensions, browsers and phisical abilities.
